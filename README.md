# .spacemacs.d

```sh
$ brew tap d12frosted/emacs-plus
$ brew install emacs-plus
$ git clone https://github.com/syl20bnr/spacemacs --depth 1 ~/.emacs.d
$ cd ~/.emacs.d
$ git clone https://github.com/razzius/.spacemacs.d ~/.spacemacs.d
$ ln -s /Applications/Emacs.app/Contents/MacOS/Emacs /usr/local/bin/emacs
$ emacs ~/.spacemacs.d/init.el
```
